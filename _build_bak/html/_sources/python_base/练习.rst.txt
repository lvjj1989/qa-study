题一

找出小于1000的正整数中，是3或5的倍数的数，并计算其和，例如:小于10中，3或5的倍数的数有3、6、5、9，和是23

题二
--------------------------------------
求600851475143的质因数

.. tip:: 质因数（素因数或质因子）在数论里是指能整除给定正整数的质数。除了1以外，两个没有其他共同质因子的正整数称为互质。因为1没有质因子，1与任何正整数（包括1本身）都是互质。正整数的因数分解可将正整数表示为一连串的质因子相乘，质因子如重复可以指数表示。根据算术基本定理，任何正整数皆有独一无二的质因子分解式。只有一个质因子的正整数为质数。如：13195的质因数是5、7、13、29
